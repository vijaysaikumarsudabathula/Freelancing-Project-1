## For Customers: 3 Simple Steps

### Step 1: Complete Payment
- Add items → Checkout → Select Payment → Pay

### Step 2: See Success Screen with Phone Numbers
```
✓ Payment Submitted!
📞 +91 8367382095
📞 +91 9010613584
```

### Step 3: Call to Confirm
- Call one of those numbers
- Give Order ID
- Confirm amount
- Done! Your order will start processing

---

## For Admin: 3 Simple Steps

### Step 1: Check Orders Tab
- Look for **yellow highlighted orders**
- See alert: "📞 Payment Confirmations Needed"

### Step 2: Call Customer to Verify
- Find pending order (yellow background)
- Call customer using details shown
- Confirm payment was received

### Step 3: Mark as Processing
- Click green **`✓ Processing`** button
- Order automatically updates
- Customer gets tracking info

## 💡 Pro Tips

### For Customers
- **Keep Order ID handy** - You'll need it for the call
- **Have email ready** - Reference your confirmation email
- **Call within 1 hour** - Faster processing
- **Be specific** - Mention exact amount paid

### For Admin
- **Check Orders daily** - Don't miss pending payments
- **Verify before updating** - Only update after phone confirmation
- **Use green button** - Makes payment confirmation obvious
- **Keep notes** - Document all verbal confirmations

### For Vendors
- **Keep phone line ready** - During business hours
- **Have payment info** - Check if payment received before confirming
- **Be professional** - Short, efficient calls
- **Update immediately** - Don't delay status updates

## ⚡ Common Questions

**Q: Why do I need to call?**
A: Manual verification prevents fraud and confirms payment.

**Q: What if I can't reach the number?**
A: Try the other number - both work, multiple people answer.

**Q: How long until my order ships?**
A: 1-2 business days after payment confirmation.

**Q: Will my email be shared?**
A: Only used for order confirmations - never shared.

**Q: What if payment failed?**
A: We'll let you know during the call and help you retry.

**Q: Can I check order status online?**
A: Yes, check your customer dashboard for tracking.

## 🚨 Troubleshooting

### Problem: Don't see phone numbers on success screen
- Solution: Refresh page or clear browser cache
- Solution: Make sure payment processing completed
- Solution: Try on different browser

### Problem: Can't find order in admin dashboard
- Solution: Check Orders tab is selected
- Solution: Look for yellow-highlighted orders
- Solution: Refresh page

### Problem: Processing button not working
- Solution: Check internet connection
- Solution: Try refreshing admin dashboard
- Solution: Try different browser

### Problem: Didn't receive confirmation email
- Solution: Check spam/junk folder
- Solution: Verify email address is correct
- Solution: Contact admin to resend
