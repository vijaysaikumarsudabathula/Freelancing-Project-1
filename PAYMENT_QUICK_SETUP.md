## Important Notes

📌 **QR Code Best Practices**
- Ensure good lighting in QR image
- No blur or distortion
- File size under 5MB
- Clear, scannable quality

📌 **UPI ID Format**
- Must include @ symbol
- Format: `accountnumber@bankname`
- Example: `9010613584@sbi`

📌 **Mobile Testing**
- Test on phone/tablet
- Verify QR scans correctly
- Check text readability
- Test all payment apps
