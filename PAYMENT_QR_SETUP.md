# 🏪 Payment QR Code & UPI Setup Guide

## Overview
Your payment system has been enhanced with a beautiful, professional UPI payment interface. Follow these steps to set it up with your QR code and UPI ID.

---

## Step 1: Access Admin Dashboard

1. Navigate to your application
2. Go to the **Login** page
3. Login with admin credentials
4. Click on **Admin Dashboard** (gear icon or admin link)

---

## Step 2: Navigate to Payment Settings

1. In the Admin Dashboard, find **Payment Settings** option
2. Click on the **Payment Settings** or **PaymentSettings** component
3. Look for the **⚙️ Payment Settings** tab

---

## Step 3: Configure UPI Payment

### Tab: 📱 UPI Settings

#### 3.1 Enter UPI ID
- Enter your UPI ID: **`9010613584@sbi`**
- Example format: `merchant@bankname`

#### 3.2 Upload QR Code
1. Click on the **UPI QR Code** input field
2. Select the QR code image file you have (the SCAN & PAY image)
3. The system will show a preview of your QR code

#### 3.3 Save Configuration
- Click the **SAVE** button at the bottom
- You'll see a success message: ✅ Payment Configuration Saved Successfully!

---

## Step 4: Enable/Disable Payment Methods

### Tab: ⚙️ Payment Settings

Toggle the payment methods you want to enable:
- **💳 Card Payment** - Credit/Debit card payments
- **📱 UPI Payment** - UPI & Mobile wallet payments  
- **🏦 Bank Transfer** - Direct bank account transfer

---

## Step 5: Configure Bank Details (Optional)

### Tab: 🏦 Bank Details

If you want to enable bank transfer payments, configure:
- Account Holder Name
- Bank Name
- Account Number
- IFSC Code
- Branch Name

Save when done.

---

## Step 6: Set Order Limits

In the **⚙️ Payment Settings** tab, scroll down to set:
- **Minimum Order Amount** (default: ₹100)
- **Maximum Order Amount** (default: ₹100,000)

## ❓ Troubleshooting

### QR Code Not Showing
1. Check file size (max 5MB)
2. Ensure file is JPG, PNG, or WebP
3. Re-upload the file
4. Clear browser cache and refresh

### UPI ID Not Displaying
1. Make sure you entered the UPI ID in the correct format
2. Click Save again
3. Refresh the page

### Styling Issues
1. Clear your browser cache (Ctrl+Shift+Delete)
2. Hard refresh the page (Ctrl+F5)
3. Check browser console for errors
